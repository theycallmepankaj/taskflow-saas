#  TaskFlow SaaS

TaskFlow is a modern SaaS-based task management application built using React, Vite, Tailwind CSS, FastAPI, MongoDB, and Zustand.

The application allows users to:

- Register and Login securely
- Create and manage tasks
- View productivity analytics
- Access protected routes using JWT authentication
- Store authentication using Zustand + localStorage
- Logout securely
- Use a responsive modern dashboard UI

---

- Lucide React

## Backend
